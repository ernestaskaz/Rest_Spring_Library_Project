# Rest_Spring_Library_Project
Book Library Project with Spring Boot, Rest, Java.


Rest endpoints:
add book;
take book(by ID or UUID);
extend book;
search book Repository by: author, category, language, ISBN, name, availability;
delete book;

Test units found at:
